# data_class_demo

本 repo 用來整理期末報告各組員負責的資料與產出。請依照自己負責的部分查看對應檔案。

| 負責內容 | 目前狀態 | 對應位置 |

## 簡報大綱與分析方法

- [簡報大綱.md](./簡報大綱.md)：本次期末報告的修正版分析架構，包含 2023-2024 建模、2025 驗證、RFM 分群、K-means/PCA、生存分析與行銷策略規劃。
- [分群回購分析實作/測試紀錄與結果.md](./分群回購分析實作/測試紀錄與結果.md)：紀錄特徵組合 A-E、K 值測試、Top 8 組合比較、評估指標意義，以及最後選擇 `B_RFM_travel / K=3` 的原因。這份文件主要給需要理解「為什麼這樣分群」的組員參考。
- [新舊版分析方法差異比較.md](./新舊版分析方法差異比較.md)：整理舊版方法提案與新版規劃/實作的主要差異，並說明新版是否保留 K-means、Silhouette 輪廓值與其他分群評估指標。

## EDA 相關檔案

目前 repo 中的 EDA 與分群測試文件只放彙總結果、測試紀錄與圖表，沒有放原始資料、客戶識別碼或單筆投保明細。
