# data_class_demo

本 repo 用來整理期末報告各組員負責的資料與產出。請依照自己負責的部分查看對應檔案。

## 檔案導覽

| 負責內容 | 目前狀態 | 對應位置 |
| --- | --- | --- |
| 統整本專案執行流程說明 | 已更新初版 | [簡報大綱.md](./簡報大綱.md) |
| 統整 EDA | 已更新 | [EDA 資料夾](./EDA/) |
| 針對客群進行行銷規劃與文案撰寫示範 | 檔案尚未更新 | 尚未提供 |


## 簡報大綱與分析方法

- [簡報大綱.md](./簡報大綱.md)：本次期末報告的修正版分析架構，包含 2023-2024 建模、2025 驗證、RFM 分群、K-means/PCA、生存分析與行銷策略規劃。

## EDA 相關檔案

- [EDA/README.md](./EDA/README.md)：完整 EDA 分享版，包含彙總統計表與視覺化圖表。
- [EDA/topic_focused_eda.md](./EDA/topic_focused_eda.md)：針對本次報告題目整理的重點版 EDA，適合製作簡報時快速抓重點。
- [EDA/charts/](./EDA/charts/)：EDA 圖表資料夾。

## 注意事項

目前 repo 中的 EDA 檔案只放彙總結果與圖表，沒有放原始資料、客戶識別碼或單筆投保明細。
