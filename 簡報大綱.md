# 簡報大綱：旅平險回購率提升與客群再行銷策略

> 文件定位：給組員製作簡報時參考的大框架。  
> 使用方式：本檔只放「報告章節、每章要回答的問題、需呈現的重點與圖表」。逐頁可放文字與講稿範例請看 `期末簡報資訊內容草稿.md`。  
> 核心主軸：面對約 76.6% 客戶只購買一次的問題，本專案希望用歷史投保行為建立客群，找出不同客群的回購傾向與提醒時機，進一步設計差異化再行銷策略，盡可能提升下一次出遊時回來投保的機會。

## 1. 一句話主張

> 旅平險不是固定週期商品，而是跟「下一次出遊」高度綁定；因此提升回購率的關鍵，不是對所有人發同一則訊息，而是依客群旅遊型態與回購節奏，在對的時間給對的提醒。

## 2. 本次簡報的核心問題

| 問題 | 簡報中要回答的方向 |
| --- | --- |
| 為什麼這題有商業價值？ | 一次購買客戶占 76.6%，代表有大量曾投保客戶尚未被有效經營。 |
| 為什麼需要分群？ | 旅平險購買行為受旅遊目的地、天數、投保時機影響，不同客戶不適合同一套提醒。 |
| 分群是否可信？ | 只用 2023-2024 建模，2025 留作驗證，避免提前使用答案。 |
| 分群是否有策略價值？ | 三群在 2025 回購率、回購速度與旅遊型態上有方向性差異，可轉成提醒時機與文案主軸。 |
| 如何落地提升回購？ | 依客群設計第一波/第二波提醒、溝通訴求與行銷優先級，後續用 A/B test 驗證成效。 |

## 5. 目前定稿分析結果

### 5.1 資料與商業痛點

| 指標 | 數值 | 簡報意義 |
| --- | ---: | --- |
| 投保紀錄數 | 379,258 | 資料量足以支撐客戶層級分析 |
| 客戶數 | 258,979 | 可觀察同一客戶是否重複投保 |
| 一次購買客戶占比 | 76.6% | 回購率提升是明確商業痛點 |
| 回購客戶占比 | 23.4% | 已存在一群會回購的客戶，可學習其行為差異 |
| 0-3 天內投保占比 | 72.8% | 提醒時機應貼近出發與旅遊檔期 |

### 5.2 最終模型選擇

| 項目 | 定稿 |
| --- | --- |
| 建模資料 | 2023-2024 投保行為 |
| 驗證資料 | 2025 回購行為 |
| 最終特徵組合 | `B_RFM_travel` |
| 最終 K 值 | 3 |
| 使用特徵 | `recency_days`, `frequency`, `total_premium`, `avg_days`, `intl_share`, `multi_dest_share` |
| 選擇理由 | 分群品質接近最佳，且比純 RFM 更能支撐旅遊情境與差異化文案 |

### 5.3 三大客群

| 客群 | 客戶數 | 占比 | 核心輪廓 | 2025 回購率 |
| --- | ---: | ---: | --- | ---: |
| 海外日本主力客 | 133,164 | 77.1% | 海外、日本目的地占比高，規模最大 | 16.7% |
| 多目的地長天數高價值客 | 7,995 | 4.6% | 旅遊天數長、多目的地、保費較高 | 17.6% |
| 臨行前快速投保客 | 31,450 | 18.2% | 短天數、低保費、臨行前投保比例高 | 15.3% |

### 5.4 2025 驗證重點

| 指標 | 結果 | 簡報解讀 |
| --- | --- | --- |
| 整體 2025 回購率 | 16.5% | 作為不分群時的基準 |
| 海外日本主力客 | 16.7%，lift 1.01x | 規模最大，適合季節性大眾提醒 |
| 多目的地長天數高價值客 | 17.6%，lift 1.07x | 回購傾向與客戶價值較高，策略優先級最高 |
| 臨行前快速投保客 | 15.3%，lift 0.93x | 回購較弱，重點是降低投保摩擦與出發前提醒 |
| log-rank test | 三組 pairwise p-value 皆 < 0.01 | 回購曲線差異具有統計顯著性，但仍不代表因果 |

## 7. 限制與表述原則

| 限制 | 報告中應如何表述 |
| --- | --- |
| 非因果分析 | 說「此群在 2025 呈現較高回購率」，不要說「此群造成回購」。 |
| 回購率差異有限 | 說「有方向性差異，且 KM 曲線差異顯著」，不要誇大成精準預測。 |
| 右設限 | 未在 2025 回購不代表永遠不回購，因此用 KM 曲線處理觀察期結束問題。 |
| 分群是策略輔助 | 最終是否能提升回購率，仍需透過 A/B test 或行銷實驗驗證。 |
| 特徵限制 | 目前只使用投保行為與旅遊型態，後續可加入通路、活動曝光、推播紀錄。 |

建議使用語句：

- 「提升回購的關鍵，是在下一次出遊前被想起。」
- 「分群提供再行銷優先級與提醒情境，不是直接預測每個人的購買日期。」
- 「2025 驗證顯示三群回購表現有方向性差異，可作為差異化策略的基礎。」
- 「後續需透過 A/B test 驗證提醒時機與文案是否真正帶動回購率提升。」

## 8. 結尾建議

最後報告不要只說「我們分成三群」，而要收斂到回購率提升：

> 面對超過七成客戶只購買一次的問題，本研究用歷史投保行為找出三個可經營客群，並透過 2025 回購驗證設計差異化提醒策略，讓旅平險再行銷從大量發送走向依旅遊節奏精準經營。
